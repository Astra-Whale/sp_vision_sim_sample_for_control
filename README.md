# sp自瞄输出模拟器
如题，一个应用了sp25自瞄云台规划器的自瞄输出模拟器。可用于采样曲线，帮助电控云台调参

**版本**：v1.0

## 功能
暂时只提供yaw输出模拟

## 配置和输出
配置区在代码最开头醒目处。输出一个txt文件到工作目录下。

**注意单位** 由于特殊约定，输出结果混合了degree和rad/s，请一定检查清楚这是否符合你们的约定

## 使用方法

配置完成后直接运行。

务必观察输出图像第一行，确认目标曲线和生成曲线大体重合。如果生成曲线上偏，说明搜索区间过小（来不及回头找下一块装甲板），反之过大（回头时间太长）。可以在配置区调整云台最大角加速度作用时间搜索范围。直到重合
